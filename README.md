## STRANGE STAR

![STRANGE STAR](screenshots/strange-star.png "STRANGE STAR")

### Description

Destroy Strange Star by Dashing spacescraft at high speed.

### Features

 - Dashing 
 - Unexpected Strange Stars

### Controls

Keyboard:
 - WASD to move
 - SPACE to Dash while moving
